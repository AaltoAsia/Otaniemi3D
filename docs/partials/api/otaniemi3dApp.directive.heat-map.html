heat-map
directive in module otaniemi3dApp

Description

Display heat map of a provided floor plan.

Usage

as element:
<heat-map
       floorplan="{Object}"
       selected-room="{Object}"
       sensor-type="{Object}">
</heat-map>

Parameters

ParamTypeDetails
floorplanObject

Floorplan that is viewed in the heat map.

selectedRoomObject

Room that is highlighted.

sensorTypeObject

Sensor type whose values are used in coloring the heat map
