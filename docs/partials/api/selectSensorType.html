selectSensorType

Description

Select current sensor type. Heat map is then colored by that sensor's values.

Usage

selectSensorType(sensor);

Parameters

ParamTypeDetails
sensorObject
  • Sensor object with following properties:
  • text - {string} - Text that is displayed on sensor selector.
  • name - {string} - Sensor's name.
  • icon - {string} - Url to sensor's icon.
